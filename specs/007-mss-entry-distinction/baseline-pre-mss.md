## 結論

- **SC-002 達標（非零 delta）**：5 檔中 3 檔（0050 / 00878 / 00631L）在交易數與/或報酬出現可量測差異——與 spec 002 `baseline-pre-fvg.md` 的 **T014 零 delta 成明確對照**。根因（`mss ⊆ bos` 使 MSS 進場虛設）已由 007 的獨立反轉進場解除。
- **SC-005**：long-only，回測交易紀錄不存在多空並存（反轉進場僅在 flat 開多；看跌反轉做空原為 BLOCKED-003，已由 spec 003 台指期做空解封）。
- **T019 回歸錨點**：`mss_reversal_entry=False` 於所有標的的 MSS 反轉進場數 = 0，等同 007 前進場；作為破壞性變更後的回歸基準（取代原 spec 001 位元一致錨點）。
- 2330/00919 無變化：該區間無「通過反轉 profile」的看漲反轉，屬正常（反轉訊號稀疏且需位移/結構條件）。

**注意（非邊際主張）**：本表僅證明「MSS 品質現在會影響回測 P&L」的機制成立，樣本小、未做顯著性檢定；不宣稱反轉進場改善績效。反轉 profile 放寬順勢確認 `{'trend'}` 與 200MA regime，但**保留三關價**（`close>mid_price`，spec 003 空頭防線；research D6 修訂）。實測：由「放寬 `{'trend','global'}`」收緊為「保留三關價」後，上表數字**完全相同**——那 3 筆反轉進場本就在中關價之上，故收緊零成本、防守更嚴。
